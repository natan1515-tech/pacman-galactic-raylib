# Pac-Man Galactic — Raylib Edition

Versão atualizada com tela inicial cinematográfica, tema galáctico, inimigos andando pelo mapa, movimento mais controlado e novos poderes.

## Como rodar no Mac

```bash
brew install raylib pkg-config
make
./pacman-raylib
```

## Controles

- **ENTER**: iniciar
- **WASD** ou **setas**: mover
- **E**: Pulso da Força
- **P**: pausar
- **R**: reiniciar
- **ESC**: voltar ao menu/sair

## Novidades desta versão

- Tela inicial com imagem dentro da pasta `assets/`.
- Inimigos saem da base e andam pelo mapa.
- Movimento mais lento e confortável.
- Partículas ao coletar cristais e poderes.
- Tremor de tela em eventos importantes.
- Power-ups extras que aparecem no mapa:
  - **Escudo**: repele inimigos próximos.
  - **Dash**: aumenta a velocidade por alguns segundos.
  - **Freeze**: congela os inimigos temporariamente.
  - **x2**: dobra os pontos coletados por tempo limitado.
- Pulso da Força com a tecla **E**, deixando vilões vulneráveis.
