# 🎮 Pacman Galactic Raylib

Uma versão moderna do clássico Pac-Man desenvolvida em C utilizando a biblioteca Raylib.

## 🚀 Sobre o Projeto

Pacman Galactic Raylib transforma o clássico jogo Pac-Man em uma experiência visual inspirada em ficção científica, com interface gráfica, efeitos visuais e poderes especiais.

## 🎯 Funcionalidades

- Interface gráfica com Raylib
- Sistema de pontuação
- Inimigos com comportamentos diferentes
- Power-ups especiais
- Tela inicial personalizada
- Sistema de partículas
- HUD dinâmica

## 🕹️ Controles

| Tecla | Função |
|---------|---------|
| ↑ ↓ ← → | Movimentação |
| E | Pulso da Força |
| R | Reiniciar |
| ESC | Menu |

## 🛠️ Tecnologias

- Linguagem C

- Grafos

- MVC
